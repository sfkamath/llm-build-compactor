# History Notes

This file is preserved as historical context from earlier Maven suppression experiments.

It does **not** describe the current implementation.

## What Changed Since These Notes

The current Maven path is extension-based and is documented in:

- `docs/maven-design.md`
- `docs/maven-extension-model.md`

The current Gradle path is documented in:

- `docs/gradle-design.md`

## Why This File Still Exists

These notes are still useful as a record of failed or partial approaches that were explored before the current design stabilized, including:

- `maven.config`-based suppression
- plugin-only suppression
- hybrid suppression experiments
- early thoughts about stream filtering

If you are trying to understand how the project works **today**, do not use this file as the source of truth.
